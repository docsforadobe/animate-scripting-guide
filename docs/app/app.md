# Application object

`app`, `an`, `animate`, `flash`, `fl`

#### Description

Provides access to objects and application settings within Animate.

The single global object is always available by its name, `app`, `an`, `animate`, `flash`, or `fl`.

---

## Attributes

### App.version

`app.version`

#### Description

The version of Animate.

#### Type

String; read-only.
