Document object
==================

``document``

**Description**
Provides access to objects and document settings within Animate.
The single global object is always available by its name, **document**.



----

==========
Attributes
==========

.. _document.name:

name
*********************************************

``document.name``

**Description**

Name of the active document.

**Type**

String; read-only.
